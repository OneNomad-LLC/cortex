# Security Advisories

This file documents all security vulnerabilities disclosed against
`@onenomad/przm-cortex` and the `przm-cortex` Docker image, in
reverse-chronological order.

Mean-time-to-patch (MTTP) is computed from the date the issue was privately
reported to the date a fix or documented mitigation shipped in a release.

## Format

Each entry follows this structure:

```
## CVE-YYYY-NNNNN — <short title>

| Field          | Value                         |
|----------------|-------------------------------|
| Date reported  | YYYY-MM-DD                    |
| Date fixed     | YYYY-MM-DD                    |
| MTTP           | N days                        |
| CVSS score     | X.Y (Critical/High/Medium/Low)|
| Affected       | vX.Y.Z – vX.Y.Z               |
| Fixed in       | vX.Y.Z                        |
| Reporter       | Name / handle (if consented)  |

### Description

Brief description of the vulnerability and its impact.

### Mitigation

Steps operators can take if they cannot upgrade immediately.

### References

- GitHub Advisory: https://github.com/advisories/GHSA-...
- NVD: https://nvd.nist.gov/vuln/detail/CVE-YYYY-NNNNN
```
